2 RとRStudioの導入

2.1 Rのインストール

RはCRAN (Comprehensive R Archive Network) から無料でダウンロードできます。

  1. CRAN にアクセス
  2. お使いのOS（Windows/Mac/Linux）を選択
  3. 最新版をダウンロードしてインストール

2.2 RStudioのインストール

RStudioは、Rを使いやすくする統合開発環境（IDE）です。

  1. RStudio Desktop にアクセス
  2. 無料版をダウンロード
  3. インストール後、RStudioを起動

2.3 RStudioの画面構成

RStudioは4つのペインで構成されています：

  • Source（左上）：スクリプトエディタ
  • Console（左下）：コマンド実行と結果表示
  • Environment（右上）：変数とデータの確認
  • Files/Plots（右下）：ファイル管理と図の表示

6.3 パイプ演算子（%>%）の活用

コードを表示 
# パイプを使わない場合（読みにくい）
result1 <- arrange(
  mutate(
    filter(exam_data, math > 70),
    total = math + english + science
  ),
  desc(total)
)

# パイプを使う場合（読みやすい）
result2 <- exam_data %>%
  filter(math > 70) %>%
  mutate(total = math + english + science) %>%
  arrange(desc(total))

# 結果は同じ
all.equal(result1, result2)
[1] TRUE

# RとRStudioの導入

## Rのインストール

RはCRAN (Comprehensive R Archive Network) から無料でダウンロードできます。

1. [CRAN](https://cran.r-project.org/) にアクセス
2. お使いのOS（Windows/Mac/Linux）を選択
3. 最新版をダウンロードしてインストール

## RStudioのインストール

RStudioは、Rを使いやすくする統合開発環境（IDE）です。

1. [RStudio Desktop](https://posit.co/download/rstudio-desktop/) にアクセス
2. 無料版をダウンロード
3. インストール後、RStudioを起動

## RStudioの画面構成

RStudioは4つのペインで構成されています：

- **Source（左上）**：スクリプトエディタ
- **Console（左下）**：コマンド実行と結果表示
- **Environment（右上）**：変数とデータの確認
- **Files/Plots（右下）**：ファイル管理と図の表示

exam_data <- data.frame(

exam_data

# filter：行の絞り込み
exam_data %>%
  filter(math > 75)

# mutate：新しい列の作成
exam_data %>%
  mutate(
    total = math + english + science,
    average = total / 3
  )

# arrange：並び替え
exam_data %>%
  mutate(total = math + english + science) %>%
  arrange(desc(total))

# group_by & summarize：グループ集計
exam_data %>%
  mutate(
    average = (math + english + science) / 3,
    grade = ifelse(average >= 75, "A", "B")
  ) %>%
  group_by(grade) %>%
  summarize(
    n = n(),
    mean_score = mean(average),
    sd_score = sd(average)
  )
