# 空間ネットワーク重ね合わせ：プロンプトとステップ

実データが無い場合でも、パッケージ内の都市データやシミュレーションで、地図（都道府県ポリゴン）上にポイントとネットワークを重ねる例を作れます。以下は、うまく動くプロンプトと手順のテンプレートです。

## ヒアリング項目（最小）

- 背景地図の単位: 都道府県/市区町村/キャンパス敷地など
- 点データの意味: 学校/学習者/教室/イベント地点
- ネットワークの構成: 距離しきい値 or k近傍（k=?） or 既存エッジ
- 出力: 図（地図重ね）、中心性の表（上位N）

## コア・プロンプト（テンプレート）

```
Rのsf/igraph/ggplot2を使って、都道府県ポリゴンを背景に、日本の主要都市ポイント（maps::world.cities）からk近傍ネットワーク（k=4）を作成し、
1) 地図上にエッジとポイント（サイズ=degree）を重ねる図、
2) 中心性（degree, betweenness）の上位10件をgt表で、
を出力するコードを、再現可能に提示してください。都道府県ポリゴンは jpndistrict/rnaturalearth/geodata の順で取得できる関数で用意し、座標はWGS84(4326)で揃え、距離計算は3857へ一時的に変換してください。
```

## 手順チェックリスト

- 背景ポリゴン取得（`load_pref_sf()`）のフォールバック順とCRSを確認
- 点データの抽出（maps::world.cities）と上位Nの選択基準（人口など）
- 距離行列→k近傍→無向化（重複除去）
- エッジをLINESTRINGにして `geom_sf()` で重ね描き
- `st_join()` で所属都道府県を付与し、gt表に含める
- 出力先（GeoJSON/CSV）の保存パスとファイル名

## 追加の発展ネタ

- 距離しきい値ネットワーク（孤立の扱い）とkNNの比較図を併記
- pref_sfのうえで地域別の中心性分位点をコロプレス（集約して表示）
- spdepのnb/listwに変換し、Moran's I などの空間自己相関と比較
