# メタ分析：プロンプトとステップ

`metafor` を中心に、SLA/教育研究のメタ分析をスムーズに回すためのプロンプト集です。

## まず確認するヒアリング項目

- 効果量タイプ：SMD, logOR, Fisher z など（生データ/集計データ）
- モデル：固定効果/ランダム効果（既定: ランダム, REML）
- 非独立性：同一研究内の複数効果あり？→ RVE/多層モデル検討
- 異質性の許容範囲と事前仮説（モデレーター候補）
- 出版バイアス評価の要否（ファンネル、Egger、trim-and-fill など）
- レポート形式（表: forest/summary、図: funnel、付録: データ一覧）

## 効果量の基礎（即決ガイド）

- 連続: 同一尺度→MD、異尺度混在→SMD（Hedges g; `measure="SMDH"`）
- 二値: log(OR)/log(RR)（報告は指数戻し）、RDも選択肢
- 相関: Fisher z（`measure="ZCOR"`；報告時は`tanh(z)`でrに戻す）
- 向き: 「正=介入有利」などを明記し、コードで統一

`metafor::escalc` の主な指定
- MD/SMD: `m1i, sd1i, n1i, m2i, sd2i, n2i`
- OR/RR/RD: `ai, bi, ci, di`（2x2 のカウント）
- Fisher z: `ri, ni`

## コア・プロンプト（テンプレート）

```
あなたはRのメタ分析アシスタントです。`metafor` を用い、効果量の算出→モデル適合→異質性→出版バイアス→メタ回帰の順で、再現可能な分析コードを提示してください。

条件:
- データ: <effects.csv>（列: study_id, yi, vi もしくは各統計量）
- モデル: ランダム効果 (REML)
- 出力: モデル要約, forest図, funnel図, Egger検定

追加:
- モデレーター: <例: 教示法 (factor), 学習者年齢 (numeric)>
- 非独立性: <あり/なし>（ありの場合はRVEの概説も）
```

## ステップ・チェックリスト（実務）

- 効果量の向き（ポジティブ=有利の定義）を統一
- 研究間重みの確認（逆分散重み、測定誤差など）
- outlier/影響点の確認（`influence()`）
- I^2 と tau^2 の報告（推定法を明記）
- モデレーターの多重検定と事前仮説の整合性
- PRISMAフロー、検索式、除外基準などの透明性確保

## 追加プロンプト（状況別）

- 「2群比較の平均とSDからHedges gを計算し、ランダム効果モデルで統合、森図を作成してください」
- 「相関係数rを Fisher z に変換して統合し、最終結果をrに戻して報告してください」
- 「研究内の複数効果があるためRVEを用いた頑健な推定（clubSandwich）を併記してください」
