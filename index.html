はじめに

本書は、英語教育および第二言語習得（SLA）研究に携わる大学院生・研究者を対象に、統計解析の理論と実践をRで学ぶためのガイドです。研究論文で頻出する手法を体系的に取り上げ、すべてのコードを実行可能な形で提供しています。

本書の特徴

実践的 英語教育・SLA研究の具体的なデータと研究課題を用いて解説
再現可能 すべてのRコードが実行可能。コピー&ペーストですぐに試せる
体系的 t検定からメタ分析まで、基礎から応用へ段階的に構成
幅広い 回帰分析、IRT、ベイズ推定、時系列、空間分析、ネットワーク分析まで網羅

対象読者

  • 英語教育・第二言語習得研究に携わる大学院生・研究者
  • Rを用いた統計解析を基礎から学びたい方
  • 言語テスト理論（IRT・ラッシュモデル・一般化可能性理論）に関心のある方
  • 時系列データや地理空間データの分析手法を知りたい方

構成

第I部：統計モデルの基礎

t検定、効果量、分散分析（ANOVA）、MANOVA、判別分析を扱います。カテゴリカル変数のコーディング方法についても詳しく解説します。

第II部：回帰分析

重回帰分析、交互作用の解釈（simple slope analysis）、一般化線形モデル（GLM）、一般化加法モデル（GAM）、混合効果モデル、2値ロジスティック回帰を扱います。

第III部：ベイズ統計と機械学習

brmsを用いたベイズ推定の基礎と、決定木・ランダムフォレスト・XGBoostによる予測分析、ROC曲線を用いたモデル評価を扱います。

第IV部：多変量解析

因子分析、主成分分析、コレスポンデンス分析、多次元尺度法、一般化可能性理論（G理論）、項目応答理論（IRT）、ラッシュモデル、拡張ラッシュモデルを扱います。

第V部：時系列解析

ARIMA、介入時系列分析（ITS）、音声波形の時系列処理を扱います。

第VI部：地理空間データ

sfパッケージによる地図作成の入門から、空間自己相関・空間回帰・施策評価の応用まで扱います。

第VII部：ネットワーク分析

学習者間・変数間の関係構造を可視化し、中心性指標やコミュニティ検出の手法を扱います。

第VIII部：メタ分析

研究成果の統合、異質性の評価、出版バイアスの検出、メタ回帰を扱います。

必要な環境

  • R（最新版を推奨）：https://cran.r-project.org/
  • RStudio（推奨）：https://posit.co/download/rstudio-desktop/

各章で使用するパッケージは章の冒頭で読み込んでいますが、以下のコードで主要なパッケージを一括インストールできます。

コード 
# 主要パッケージの一括インストール
required_packages <- c(
  # データ操作・可視化
  "tidyverse", "gt", "gtExtras", "patchwork", "GGally",

  # 統計解析（基礎）
  "car", "effectsize", "pwr", "emmeans", "psych",

  # 回帰・混合効果モデル
  "lme4", "lmerTest", "lavaan",

  # IRT・ラッシュモデル
  "mirt", "eRm", "TAM", "mokken",

  # ベイズ推定
  "brms", "rstanarm", "bayesplot",

  # ネットワーク分析
  "igraph",

  # メタ分析
  "metafor", "clubSandwich", "meta", "netmeta"
)

install.packages(required_packages)
