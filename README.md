# Rによる統計解析実戦ガイド（英語教育・第二言語習得向け）

本リポジトリは、Quarto Book を用いて作成した学習サイトです。英語教育・第二言語習得（SLA）研究の現場で役立つ統計手法を、Rコードとともに実践的にまとめています。

## 収録内容（章立て）

- 第I部：統計モデルの基礎（t検定、ANOVA など）
- 第II部：回帰分析（重回帰、GLM、混合効果モデル など）
- 第III部：ベイズ統計と機械学習（入門）
- 第IV部：多変量解析（因子分析、G理論、IRT/ラッシュ、拡張ラッシュ など）
- 第V部：時系列データ分析（基礎・発展・音声の3章構成）
- 第VI部：地理空間データ（入門・応用）
- 第VII部：ネットワーク分析
- 第VIII部：メタ分析
- 第IX部：因果推論とクラスタリング

トップページと全章は `_quarto.yml` の `book.chapters` に基づいてレンダリングされます。

## 必要環境

- R（最新版推奨）
- Quarto（https://quarto.org/）
- 代表的なRパッケージ：tidyverse, mirt, eRm, TAM, mokken, lavaan, psych など

## ビルド方法

```bash
# サイトのプレビュー（ホットリロード）
quarto preview

# 書き出し（_book ディレクトリに出力）
quarto render
```

PDF/EPUB も `_quarto.yml` の設定に従って生成できます（LaTeX 環境が必要な場合があります）。

### HTML ビルドの実務メモ

このプロジェクトは Dropbox 配下にあり、`quarto render --to html` の最後で `_book/` に大量の HTML/図版資産を書き戻す処理が数分かかることがあります。日常的なローカル確認では、まず `/tmp` にレンダリングする方が安定します。

```bash
# 高速なローカル HTML ビルド（/tmp に出力）
bash _scripts/render_book_html.sh

# /tmp にレンダリングしたあと、_book/ と index.html も更新
bash _scripts/render_book_html.sh --sync
```

`quarto render` 自体も引き続き使えますが、Dropbox 配下では `[26/26]` の後に出力同期が長く見えることがあります。`_scripts/render_book_html.sh` は Quarto の本体処理を維持したまま、重い最終書き出しだけをローカル一時ディレクトリへ逃がすための補助スクリプトです。

## スタイルと一貫性

- サイト全体のテーマは Quarto の Bootswatch テーマ `journal` を採用しています（`_quarto.yml`）。
- 図のラベル・タイトルは英語で統一し、本文で日本語の解説を添えています（OS/フォント差による文字化け回避）。
- 効果量の符号は「正=介入有利」を原則に統一しています（章冒頭に明記）。
- 各章では日本語フォント・目次・セクション番号等を統一しています。章側で追加のスタイルを定義している箇所もありますが、基本方針は `journal` に準拠します。

## 構成ファイルと主要ファイル

- `_quarto.yml`：書籍全体の構成と書式（テーマ、目次、出力形式）
- `index.qmd`：書籍トップページ（はじめに）
- `*_*.qmd`：各章の本文（Rコードと解説）
- 章ファイル（.qmd）はルート直下に配置
- `prompts/`：章ごとのプロンプトテンプレートと手順メモ

## フォルダ整理の方針（今後）

- 章はルート直下に統一しました（過去に一時的に `chapters/` を使用していた名残がある場合があります）
- 図表や補助データは `data/` と将来的な `assets/` 配下に整理
- Quartoのビルド成果物やキャッシュは `.gitignore` で除外（`_book/`, `*_cache/`, `*_files/` など）
