7 はじめに

t検定は1つの目的変数（連続値）に対して1つの説明変数（2水準のカテゴリカル変数）を扱う線形モデルです。ANOVAは1つの目的変数に対して複数の説明変数を扱います。MANOVA（多変量分散分析）は複数の目的変数を同時に扱い、それらの共分散構造を踏まえて群差を検討します。ここでは語彙テストの 再生（Recall） と 再認（Recognition） を目的変数とし、Treatment（Glossing / Writing） と Time（Immediate / Delayed） を要因として説明します。

8 用語

  • 単回帰・重回帰：目的変数が1つで、説明変数が連続の場合。
  • ANOVA：目的変数が1つで、説明変数がカテゴリカルの場合。
  • ANCOVA：目的変数が1つ、カテゴリカルな要因に加えて共変量（連続）がある場合。
  • MANOVA：目的変数が複数で、説明変数がカテゴリカルの場合。
  • MANCOVA：複数目的変数＋カテゴリカル要因＋共変量。

9 予備的な統計概念

9.1 タイプIエラーとファミリーワイズエラー率

帰無仮説が真であるのにそれを棄却してしまう確率。標準的には \(\alpha=0.05\) が用いられます。多重比較が増えるほど家族内誤差率（family-wise \(\alpha\)）や実験内誤差率（experiment-wise \(\alpha\)）が膨らみます。

9.2 Bonferroniの不等式

\(k\) 個の独立検定を名目上 \(\alpha\) で行うと、全体の誤棄却確率は概ね \(k\alpha\) に近づきます。したがって多重比較では調整（例：Bonferroni, Holm など）が重要です。

9.3 効果量

MANOVAに関連する効果量は、ウィルクスのラムダ \(\Lambda\) から \(1-\Lambda\) を効果量類似として報告する慣習や、単変量追跡での \(\eta^2\) / 偏\(\eta^2\) などがあります（厳密には加法性を持ちません）。

9.4 検定力

検定力（power）は、真に効果があるときにそれを検出できる確率。サンプルサイズ、効果量、名目上の \(\alpha\) に依存します。

10 MANOVAの概要

MANOVAはSSCP（平方和・外積）行列に基づく一般化分散の比較により、要因が目的変数の線形結合に与える影響を検定します。代表的な統計量は Pillai、Wilks、Hotelling–Lawley、Roy です。小標本や前提違反への頑健性の観点では Pillai がしばしば推奨されます。

11 データの作成

コードを表示 
# Generate sample data
n_participants <- 50
n_items <- 30
treatments <- c("Glossing", "Writing")
tests <- c("Recall", "Recognition")
times <- c("Immediate", "Delayed")

# Create data frame
dat <- expand.grid(
  Participant = 1:n_participants,
  Item        = 1:n_items,
  Treatment   = treatments,
  Test        = tests,
  Time        = times,
  KEEP.OUT.ATTRS = FALSE
)

# Assign treatments to items
dat <- dat[(dat$Treatment == "Glossing" & dat$Item <= 15) |
           (dat$Treatment == "Writing"  & dat$Item >  15), ]

# Generate scores
dat$Score <- ifelse(
  dat$Treatment == "Glossing",
  rbinom(nrow(dat), 1, ifelse(dat$Test == "Recognition", 0.5, 0.2)),
  rbinom(nrow(dat), 1, ifelse(dat$Test == "Recognition", 0.9, 0.5))
)

# Adjust for time effect
dat$Score <- ifelse(dat$Time == "Immediate",
                    pmin(dat$Score + sample(0:1, nrow(dat), replace = TRUE), 1),
                    dat$Score)

# Summarize data
dat_summarized <- dat |>
  mutate(across(c(Participant, Item, Treatment, Test, Time), as.factor)) |>
  group_by(Participant, Treatment, Test, Time) |>
  summarise(Score = sum(Score), .groups = "drop")

# Create wide format for MANOVA
wide_scores <- dat_summarized |>
  pivot_wider(names_from = Test, values_from = Score, names_prefix = "Score_") |>
  mutate(across(c(Participant, Treatment, Time), as.factor))

12 MANOVAの前提（検定・可視化）

12.1 多変量正規性

複数変数が共同で正規分布に従うかを検証します。個別に正規でも連合が正規とは限りません。

# Prepare data for multivariate normality tests
data_for_mvn <- wide_scores |> 
  dplyr::select(Score_Recall, Score_Recognition)

# 確実に動作する標準的な方法を使用

shapiro_recognition <- shapiro.test(wide_scores$Score_Recognition)

p <- ncol(data_for_mvn)
cov_matrix <- cov(data_for_mvn)

# マハラノビス距離の計算

12.2 共分散行列の等質性

群間で分散だけでなく共分散も等しいことを確認します。BoxのMは有意になりやすいため解釈に注意し、補助的に各変数の等分散性（Bartlett）も確認します。

コードを表示 
# Create interaction group for testing
wide_scores$Group <- interaction(wide_scores$Treatment, wide_scores$Time)

# Box's M test
box_m_result <- biotools::boxM(
  data_for_mvn, 
  grouping = wide_scores$Group
)

# Bartlett tests for each variable
bartlett_recall <- bartlett.test(Score_Recall ~ Group, data = wide_scores)
bartlett_recognition <- bartlett.test(Score_Recognition ~ Group, data = wide_scores)

# Create summary table
homogeneity_results <- tibble(
  Test = c("Box's M", "Bartlett (Recall)", "Bartlett (Recognition)"),
  Statistic = c(
    as.numeric(box_m_result$statistic),
    bartlett_recall$statistic[[1]],
    bartlett_recognition$statistic[[1]]
  ),
  df = c(
    as.numeric(box_m_result$parameter),
    bartlett_recall$parameter[[1]],
    bartlett_recognition$parameter[[1]]
  ),
  `p-value` = c(
    box_m_result$p.value,
    bartlett_recall$p.value,
    bartlett_recognition$p.value
  ),
  Decision = c(
    ifelse(box_m_result$p.value < 0.05, "Heterogeneous", "Homogeneous"),
    ifelse(bartlett_recall$p.value < 0.05, "Heterogeneous", "Homogeneous"),
    ifelse(bartlett_recognition$p.value < 0.05, "Heterogeneous", "Homogeneous")

# はじめに

t検定は1つの目的変数（連続値）に対して1つの説明変数（2水準のカテゴリカル変数）を扱う線形モデルです。ANOVAは1つの目的変数に対して複数の説明変数を扱います。**MANOVA**（多変量分散分析）は**複数の目的変数**を同時に扱い、それらの**共分散構造**を踏まえて群差を検討します。ここでは語彙テストの **再生（Recall）** と **再認（Recognition）** を目的変数とし、**Treatment（Glossing / Writing）** と **Time（Immediate / Delayed）** を要因として説明します。

# 用語

-   **単回帰・重回帰**：目的変数が1つで、説明変数が連続の場合。
-   **ANOVA**：目的変数が1つで、説明変数がカテゴリカルの場合。
-   **ANCOVA**：目的変数が1つ、カテゴリカルな要因に加えて共変量（連続）がある場合。
-   **MANOVA**：**目的変数が複数**で、説明変数がカテゴリカルの場合。
-   **MANCOVA**：複数目的変数＋カテゴリカル要因＋共変量。

# 予備的な統計概念

## タイプIエラーとファミリーワイズエラー率

帰無仮説が真であるのにそれを棄却してしまう確率。標準的には $\alpha=0.05$ が用いられます。多重比較が増えるほど**家族内誤差率（family-wise** $\alpha$）や**実験内誤差率（experiment-wise** $\alpha$）が膨らみます。

## Bonferroniの不等式

$k$ 個の独立検定を名目上 $\alpha$ で行うと、全体の誤棄却確率は概ね $k\alpha$ に近づきます。したがって多重比較では**調整**（例：Bonferroni, Holm など）が重要です。

## 効果量

MANOVAに関連する効果量は、ウィルクスのラムダ $\Lambda$ から $1-\Lambda$ を効果量類似として報告する慣習や、単変量追跡での $\eta^2$ / 偏$\eta^2$ などがあります（厳密には加法性を持ちません）。

## 検定力

検定力（power）は、真に効果があるときにそれを検出できる確率。サンプルサイズ、効果量、名目上の $\alpha$ に依存します。

# MANOVAの概要

MANOVAは**SSCP（平方和・外積）行列**に基づく一般化分散の比較により、要因が目的変数の線形結合に与える影響を検定します。代表的な統計量は **Pillai**、**Wilks**、**Hotelling–Lawley**、**Roy** です。小標本や前提違反への頑健性の観点では Pillai がしばしば推奨されます。

# Check if Treatment effect exists in the stats matrix

  # Use first non-residual row as fallback

  if(length(non_residual_rows) > 0) {

  } else {
    pillai_stat <- 0.3  # Default value
    print("Could not extract Pillai's trace, using default value")
  }
}

  sig.level = 0.05

## 結論

-   MANOVA は複数の目的変数を同時に扱い、相関を踏まえて群差を検討します。
-   Pillai / Wilks / Hotelling-Lawley / Roy を総合して解釈し、必要に応じて単変量や判別分析で追跡します。
-   前提（多変量正規性・共分散等質・独立性）を確認します。
-   判別分析により、群を最もよく分離する変数の線形結合を特定できます。
-   事後検定により具体的な群間差を明確にできます。
-   効果量とパワー分析により実質的な意味と研究の妥当性を評価します。
