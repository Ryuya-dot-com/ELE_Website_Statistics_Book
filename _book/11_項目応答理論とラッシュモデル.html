英単語のテストは「単語力」を測定しています。しかし、実際のテストでは「以下の英単語の意味として最も適切なものをa〜dから1つずつ選びなさい。1. apple、2. pear、3. fig …」のように、個別の単語が出題されます。そのため「単語力」のような目に見えない能力を、目に見えるテスト項目を使って測定していると言えます。ちなみに上記の答えはリンゴ、ナシ、イチジクとなります。fig (イチジク) はもしかしたら難しい項目であり、受験者の点数が低いかもしれません。

目に見えるデータは観測変数、目に見えないデータは潜在変数と呼ばれています。つまり、上記の例では観測変数は受験者から集めたテストの結果 (正解・不正解のような2値データ、平均点・分散) となり、潜在変数 (単語力) は因子分析などによって仮定されます。つまり、潜在変数それ自体は測定することはできません。

通過率\(p_j\)は以下のように計算されます：

\[ p_j = \frac{1}{N}\sum_{i=1}^{N}u_{ij} \]

つまり通過率は、項目\(j\)に正解した受験者の割合であり、項目\(j\)の平均点と解釈することもできます。

14.2.1 信頼性係数

観測変数\(x\)は潜在変数\(t\)と誤差\(e\)の和であると考えると (\(x = t + e\))、テスト得点の分散は真の得点の分散と誤差の分散の単純な和で表されます：

\[\sigma_{x}^2 = \sigma_{t}^2+\sigma_e^2\]

この分散の比ρを信頼性係数と言います：

\[\rho = \frac{\sigma_t^2}{\sigma_t^2+\sigma_e^2}=1-\frac{\sigma_e^2}{\sigma_x^2}\]

14.2.3 古典的テスト理論の限界点

Main Limitations of CTT
  1. Test-taker dependency: Item analysis results completely depend on the characteristics of the test-taker population
  2. Test dependency: Ability estimation depends on the test used
  3. Scale issues: Validity of treating ordinal scales as interval scales
  4. Measurement precision: Assumes constant measurement precision across all score ranges

19.1 テスト情報量の計算式

Test Information for Each Model
  • 1PL: \(I(\theta_i)=D^2a^2\sum_{j=1}^n p_j(\theta_i) q_j(\theta_i)\)
  • 2PL: \(I(\theta_i)=D^2\sum_{j=1}^n a_j^2 p_j(\theta_i) q_j(\theta_i)\)
  • 3PL: \(I(\theta_i)=D^2\sum_{j=1}^n \dfrac{a_j^2(p_j(\theta_i)-c_j)^2 q_j(\theta_i)}{p_j(\theta_i)(1-c_j)^2}\)

Where: \(q_j(\theta) = 1 - p_j(\theta)\), \(D = 1.7\) (normal metric) or \(D = 1.0\) (logistic metric)

24 まとめ

項目応答理論（IRT）は、古典的テスト理論（CTT）の限界を克服し、より精密な測定を可能にする現代的な心理測定理論です。主な利点として：

  1. 受験者集団に依存しない項目パラメータ
  2. 能力値ごとの測定精度の評価
  3. 異なるテスト間の等化
  4. 適応型テストの実装

通過率$p_j$は以下のように計算されます：

$$
p_j = \frac{1}{N}\sum_{i=1}^{N}u_{ij}
$$

つまり通過率は、項目$j$に正解した受験者の割合であり、項目$j$の平均点と解釈することもできます。

### 信頼性係数

観測変数$x$は潜在変数$t$と誤差$e$の和であると考えると ($x = t + e$)、テスト得点の分散は真の得点の分散と誤差の分散の単純な和で表されます：

$$\sigma_{x}^2 = \sigma_{t}^2+\sigma_e^2$$

この分散の比ρを信頼性係数と言います：

$$\rho = \frac{\sigma_t^2}{\sigma_t^2+\sigma_e^2}=1-\frac{\sigma_e^2}{\sigma_x^2}$$

## モデル式

### 1PL（ラッシュモデル）
$$p_j(\theta)=\frac{1}{1+\exp(-Da(\theta-b_j))}$$

### 2PL
$$p_j(\theta)=\frac{1}{1+\exp(-Da_j(\theta-b_j))}$$

### 3PL
$$p_j(\theta)=c_j+\frac{1-c_j}{1+\exp(-Da_j(\theta-b_j))}$$

## テスト情報量の計算式

::: {.callout-note}
## Test Information for Each Model

- **1PL**: $I(\theta_i)=D^2a^2\sum_{j=1}^n p_j(\theta_i) q_j(\theta_i)$
- **2PL**: $I(\theta_i)=D^2\sum_{j=1}^n a_j^2 p_j(\theta_i) q_j(\theta_i)$
- **3PL**: $I(\theta_i)=D^2\sum_{j=1}^n \dfrac{a_j^2(p_j(\theta_i)-c_j)^2 q_j(\theta_i)}{p_j(\theta_i)(1-c_j)^2}$

Where: $q_j(\theta) = 1 - p_j(\theta)$, $D = 1.7$ (normal metric) or $D = 1.0$ (logistic metric)
:::

# まとめ

項目応答理論（IRT）は、古典的テスト理論（CTT）の限界を克服し、より精密な測定を可能にする現代的な心理測定理論です。主な利点として：

1. **受験者集団に依存しない項目パラメータ**
2. **能力値ごとの測定精度の評価**
3. **異なるテスト間の等化**
4. **適応型テストの実装**

が挙げられます。Rには豊富なIRT分析パッケージが用意されており、研究目的に応じた柔軟な分析が可能です。
