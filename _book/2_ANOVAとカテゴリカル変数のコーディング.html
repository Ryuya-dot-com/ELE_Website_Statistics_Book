5 はじめに

本章の目的は、英語教育研究で頻出する要因計画データを、ANOVAを「回帰モデルの一形態」として一貫的に理解することです。多くの教科書ではANOVAと回帰を別章で学びますが、実際には両者は同じ一般線形モデル（General Linear Model; GLM）に属します。この観点を押さえると、カテゴリカル変数のコーディング（contrast coding）と結果解釈が大幅に明瞭になります。

本章では以下を到達目標とします。

  1. ANOVAを線形モデルとして数式レベルで説明できる。
  2. 反復測定を含むデザインで、仮定（特に球面性）を正しく説明できる。
  3. afexとemmeansを使って、主効果・交互作用・事後比較を再現可能に報告できる。
  4. Treatment / Sum / Helmert / Polynomial codingの違いを、設計行列と係数解釈の観点から説明できる。

6 1. ANOVAを回帰モデルとして捉える

6.1 1.1 共通の枠組み

ANOVAも回帰も、基本式は同じです。

\[ \mathbf{y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\varepsilon}, \quad \boldsymbol{\varepsilon} \sim N(0, \sigma^2\mathbf{I}) \]

  • \(\mathbf{y}\): 従属変数（例: テスト得点）
  • \(\mathbf{X}\): 設計行列（カテゴリカル変数はコーディングされる）
  • \(\boldsymbol{\beta}\): 係数
  • \(\boldsymbol{\varepsilon}\): 誤差

カテゴリカル予測子が\(k\)水準なら、通常\(k-1\)本のコントラスト列が必要です。どのコーディングを選ぶかで係数の意味は変わるが、同じモデル空間なら適合値や全体検定は本質的に同じ、という点が実務上重要です（Schad et al., 2020）。

6.2 1.2 F検定の意味

ANOVAのF統計量は、

\[ F = \frac{\text{MS}_{\text{effect}}}{\text{MS}_{\text{error}}} \]

であり、直感的には「効果で説明される変動」が「誤差変動」に比べて十分大きいかを見ます。2群比較では\(t^2 = F\)が成り立つため、t検定とANOVAは対立概念ではなく、同一枠組みの特殊ケースです。

7 2. 研究計画と仮定

7.1 2.1 英語教育研究での典型デザイン

英語教育研究では、次の3タイプがよく使われます。

  1. 被験者間計画（例: 指導法A群 vs B群）
  2. 被験者内計画（例: 同一学習者の直後テスト・遅延テスト）
  3. 混合計画（例: 指導法は被験者間、時点は被験者内）

本章の実装例は、最も実務的な混合計画を中心に扱います。

7.2 2.2 仮定の整理

仮定 何を要求するか 主な確認方法 注意点
独立性 観測単位間の誤差が独立 研究デザインで担保 統計手法だけでは救済不能
正規性 残差が概ね正規 QQプロット, 残差診断 大標本では頑健なことが多い
等分散性 群間で残差分散が類似 Levene検定, 残差プロット 破れが大きければロバスト法検討
球面性 反復測定要因（3水準以上）の差得点分散が等しい Mauchly検定 + GG/HF補正 2水準要因では自動的に満たされる

球面性違反時の自由度補正としては、Greenhouse-Geisser補正とHuynh-Feldt補正が古典的に用いられます。なお、複合対称性（compound symmetry）は球面性より強い仮定であり、両者を同一視しないことが重要です。

8 3. データ生成（英語教育の想定）

ここでは、次の研究設定を模したシミュレーションデータを作成します。

  • 被験者間要因: Instruction（Glossing, Writing）
  • 被験者内要因: Time（Immediate, Delayed）
  • 共変量: Proficiency
  • 従属変数: 語彙テスト得点 Score
コード 
library(tidyverse)
library(afex)
library(emmeans)
library(effectsize)

set.seed(2026)

コード 
n_participants <- 120

id_tbl <- tibble(
  Participant = factor(1:n_participants),
  Instruction = factor(rep(c("Glossing", "Writing"), each = n_participants / 2),
                       levels = c("Writing", "Glossing")),
  Proficiency = as.numeric(scale(rnorm(n_participants, mean = 2500, sd = 120))),
  u0 = rnorm(n_participants, mean = 0, sd = 4)
)

dat <- tidyr::crossing(
  Participant = id_tbl$Participant,
  Time = factor(c("Immediate", "Delayed"), levels = c("Immediate", "Delayed"))
) |>
  left_join(id_tbl, by = "Participant") |>
  mutate(
    mu = 58 +
      if_else(Instruction == "Glossing", 4, 0) +
      if_else(Time == "Delayed", -6, 0) +
      if_else(Instruction == "Glossing" & Time == "Delayed", 2, 0) +
      3 * Proficiency + u0,
    Score = round(rnorm(n(), mean = mu, sd = 5), 1)
  ) |>
  dplyr::select(-u0, -mu)

dat |>
  group_by(Instruction, Time) |>
  summarise(
    N = n(),
    Mean = mean(Score),
    SD = sd(Score),
    .groups = "drop"
  ) |>
  gt() |>
  tab_header(title = "Descriptive Statistics by Instruction and Time")

9 4. 混合要因ANOVAの実行

9.1 4.1 主効果・交互作用

コード 
fit_aov <- aov_ez(
  id = "Participant",
  dv = "Score",
  between = "Instruction",
  within = "Time",
  covariate = "Proficiency",
  factorize = FALSE,
  data = dat,
  type = 3,
  anova_table = list(es = "pes")
)

nice(fit_aov) |>
  gt() |>
  tab_header(
    title = "Mixed ANOVA (Type III)",
    subtitle = md("Dependent variable: **Score**")
  )

afexを使う利点は、Type III平方和・効果量・必要に応じた球面性補正を統一的に扱える点です。本章では効果量を偏η²（η²p）中心で報告します。教育研究の報告では、主効果だけでなく交互作用を先に確認し、交互作用が有意なら単純主効果の検討へ進む流れが実務上自然です。

11 6. カテゴリカル変数のコーディング

ここからは、ANOVAの解釈を左右するコーディングを整理します。ポイントは次の1点です。

重要

コーディングは「係数の意味」を変える。 ただし同じモデル空間なら、適合値・残差・全体的なモデル適合は本質的に変わらない。

11.1 6.1 比較用データ（3群）

コード 
set.seed(2026)

dat_c <- tibble(
  Group = factor(rep(c("Control", "Glossing", "Keyword"), each = 100),
                 levels = c("Control", "Glossing", "Keyword")),
  Score = c(
    rnorm(100, mean = 50, sd = 8),
    rnorm(100, mean = 58, sd = 8),
    rnorm(100, mean = 62, sd = 8)
  )
)

dat_c |>
  group_by(Group) |>
  summarise(Mean = mean(Score), SD = sd(Score), .groups = "drop") |>
  gt() |>
  tab_header(title = "Descriptive Statistics for Contrast Coding")

11.5 6.5 順序尺度にはPolynomial coding

コード 
set.seed(2026)

dat_ord <- tibble(
  CEFR = factor(rep(c("A1", "A2", "B1", "B2"), each = 80),
                levels = c("A1", "A2", "B1", "B2"),
                ordered = TRUE),

    rnorm(80, 48, 7),
    rnorm(80, 56, 7),
    rnorm(80, 66, 7),

contrasts(dat_ord$CEFR) <- contr.poly(4)

  1. 球面性と複合対称性を混同しない。
  2. 交互作用が有意なのに主効果だけで結論を出さない。
  3. 係数表だけで議論せず、emmeansで比較を統一する。
  4. 効果量をp値の代替ではなく補完情報として扱う。
  5. コーディングの選択理由を本文で明示する。

# はじめに

本章の目的は、英語教育研究で頻出する要因計画データを、**ANOVAを「回帰モデルの一形態」として一貫的に理解すること**です。多くの教科書ではANOVAと回帰を別章で学びますが、実際には両者は同じ一般線形モデル（General Linear Model; GLM）に属します。この観点を押さえると、カテゴリカル変数のコーディング（contrast coding）と結果解釈が大幅に明瞭になります。

本章では以下を到達目標とします。

1. ANOVAを線形モデルとして数式レベルで説明できる。
2. 反復測定を含むデザインで、仮定（特に球面性）を正しく説明できる。
3. `afex`と`emmeans`を使って、主効果・交互作用・事後比較を再現可能に報告できる。
4. Treatment / Sum / Helmert / Polynomial codingの違いを、設計行列と係数解釈の観点から説明できる。

# 1. ANOVAを回帰モデルとして捉える

## 1.1 共通の枠組み

ANOVAも回帰も、基本式は同じです。

$$
\mathbf{y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\varepsilon}, \quad \boldsymbol{\varepsilon} \sim N(0, \sigma^2\mathbf{I})
$$

- $\mathbf{y}$: 従属変数（例: テスト得点）
- $\mathbf{X}$: 設計行列（カテゴリカル変数はコーディングされる）
- $\boldsymbol{\beta}$: 係数
- $\boldsymbol{\varepsilon}$: 誤差

カテゴリカル予測子が$k$水準なら、通常$k-1$本のコントラスト列が必要です。**どのコーディングを選ぶかで係数の意味は変わるが、同じモデル空間なら適合値や全体検定は本質的に同じ**、という点が実務上重要です（Schad et al., 2020）。

## 1.2 F検定の意味

ANOVAのF統計量は、

$$
F = \frac{\text{MS}_{\text{effect}}}{\text{MS}_{\text{error}}}
$$

であり、直感的には「効果で説明される変動」が「誤差変動」に比べて十分大きいかを見ます。2群比較では$t^2 = F$が成り立つため、t検定とANOVAは対立概念ではなく、同一枠組みの特殊ケースです。

# 2. 研究計画と仮定

## 2.1 英語教育研究での典型デザイン

英語教育研究では、次の3タイプがよく使われます。

1. 被験者間計画（例: 指導法A群 vs B群）
2. 被験者内計画（例: 同一学習者の直後テスト・遅延テスト）
3. 混合計画（例: 指導法は被験者間、時点は被験者内）

本章の実装例は、最も実務的な**混合計画**を中心に扱います。

## 2.2 仮定の整理

| 仮定 | 何を要求するか | 主な確認方法 | 注意点 |
|---|---|---|---|
| 独立性 | 観測単位間の誤差が独立 | 研究デザインで担保 | 統計手法だけでは救済不能 |
| 正規性 | 残差が概ね正規 | QQプロット, 残差診断 | 大標本では頑健なことが多い |
| 等分散性 | 群間で残差分散が類似 | Levene検定, 残差プロット | 破れが大きければロバスト法検討 |
| 球面性 | 反復測定要因（3水準以上）の差得点分散が等しい | Mauchly検定 + GG/HF補正 | 2水準要因では自動的に満たされる |

球面性違反時の自由度補正としては、Greenhouse-Geisser補正とHuynh-Feldt補正が古典的に用いられます。なお、**複合対称性（compound symmetry）は球面性より強い仮定**であり、両者を同一視しないことが重要です。

# 3. データ生成（英語教育の想定）

ここでは、次の研究設定を模したシミュレーションデータを作成します。

- 被験者間要因: `Instruction`（`Glossing`, `Writing`）
- 被験者内要因: `Time`（`Immediate`, `Delayed`）
- 共変量: `Proficiency`
- 従属変数: 語彙テスト得点 `Score`

```{r}
#| label: make-data

n_participants <- 120

id_tbl <- tibble(
  Participant = factor(1:n_participants),
  Instruction = factor(rep(c("Glossing", "Writing"), each = n_participants / 2),
                       levels = c("Writing", "Glossing")),

  dplyr::select(-u0, -mu)

dat |>
  group_by(Instruction, Time) |>
  summarise(

    .groups = "drop"
```

# 4. 混合要因ANOVAの実行

## 4.1 主効果・交互作用

```{r}
#| label: mixed-anova

fit_aov <- aov_ez(
  id = "Participant",
  dv = "Score",
  between = "Instruction",
  within = "Time",
  covariate = "Proficiency",
  factorize = FALSE,
  data = dat,
  type = 3,
  anova_table = list(es = "pes")
)

nice(fit_aov) |>
  gt() |>
  tab_header(
    title = "Mixed ANOVA (Type III)",
    subtitle = md("Dependent variable: **Score**")
  )
```

`afex`を使う利点は、Type III平方和・効果量・必要に応じた球面性補正を統一的に扱える点です。本章では効果量を**偏η²（η²p）中心**で報告します。教育研究の報告では、主効果だけでなく交互作用を先に確認し、交互作用が有意なら単純主効果の検討へ進む流れが実務上自然です。

## 4.2 事後比較（推定周辺平均）

```{r}
#| label: posthoc

emm <- emmeans(fit_aov, ~ Instruction * Time)

# 時点ごとに指導法を比較
pairs(emm, by = "Time", adjust = "holm") |>
  as.data.frame() |>
  mutate(across(where(is.numeric), ~ round(., 3))) |>
  gt() |>
  tab_header(title = "Pairwise Comparisons: Instruction within Time (Holm-adjusted)")

# 指導法ごとに時点を比較
pairs(emm, by = "Instruction", adjust = "holm") |>
  as.data.frame() |>
  mutate(across(where(is.numeric), ~ round(., 3))) |>
  gt() |>
  tab_header(title = "Pairwise Comparisons: Time within Instruction (Holm-adjusted)")
```

`emmeans`では、モデルに整合的な推定周辺平均を基盤に比較を行えるため、係数表の直接比較より解釈が一貫します。多群比較では`adjust = "tukey"`や`"holm"`の選択理由を明記すると、査読での透明性が高まります。

## 4.3 効果量

```{r}
#| label: effect-size

# 全体モデルのR^2
performance::r2(lm(Score ~ Instruction * Time + Proficiency, data = dat))

# ANOVA系効果量
eta_squared(fit_aov, partial = TRUE, ci = 0.95)
omega_squared(fit_aov, partial = TRUE, ci = 0.95)
```

本章では偏η²（η²p）を第一報告指標とし、必要に応じてω²を補助的に併記します。教育・心理系では、効果量と信頼区間の併記が推奨されます。

# 6. カテゴリカル変数のコーディング

ここからは、ANOVAの解釈を左右するコーディングを整理します。ポイントは次の1点です。

::: {.callout-important}
**コーディングは「係数の意味」を変える。**
ただし同じモデル空間なら、適合値・残差・全体的なモデル適合は本質的に変わらない。
:::

## 6.1 比較用データ（3群）

```{r}
#| label: contrast-data

set.seed(2026)

dat_c <- tibble(
  Group = factor(rep(c("Control", "Glossing", "Keyword"), each = 100),
                 levels = c("Control", "Glossing", "Keyword")),
  Score = c(
    rnorm(100, mean = 50, sd = 8),
    rnorm(100, mean = 58, sd = 8),
    rnorm(100, mean = 62, sd = 8)
  )
)

dat_c |>
  group_by(Group) |>
  summarise(Mean = mean(Score), SD = sd(Score), .groups = "drop") |>
  gt() |>
  tab_header(title = "Descriptive Statistics for Contrast Coding")
```

## 6.2 Treatment coding（ダミーコーディング）

```{r}
#| label: treatment-coding

dat_treat <- dat_c
contrasts(dat_treat$Group) <- contr.treatment(3)

contrasts(dat_treat$Group) |>
  as.data.frame() |>
  tibble::rownames_to_column("Level") |>
  gt() |>
  tab_header(title = "Design Matrix: Treatment Coding")

fit_treat <- lm(Score ~ Group, data = dat_treat)

tidy(fit_treat) |>
  mutate(across(where(is.numeric), ~ round(., 3))) |>
  gt() |>
  tab_header(title = "Coefficient Table: Treatment Coding")
```

- 切片: 参照群（ここでは`Control`）の平均
- 係数: 各群と参照群との差
- 統制群が明確な介入研究で特に有用

## 6.3 Sum coding（偏差コーディング）

```{r}
#| label: sum-coding

dat_sum <- dat_c
contrasts(dat_sum$Group) <- contr.sum(3)

contrasts(dat_sum$Group) |>
  as.data.frame() |>
  tibble::rownames_to_column("Level") |>
  gt() |>
  tab_header(title = "Design Matrix: Sum Coding")

fit_sum <- lm(Score ~ Group, data = dat_sum)

tidy(fit_sum) |>
  mutate(across(where(is.numeric), ~ round(., 3))) |>
  gt() |>
  tab_header(title = "Coefficient Table: Sum Coding")
```

- 切片: 総平均（grand mean）
- 係数: 各水準が総平均からどれだけずれるか
- ANOVAの主効果解釈と整合的

## 6.4 Helmert coding

```{r}
#| label: helmert-coding

dat_helm <- dat_c
contrasts(dat_helm$Group) <- contr.helmert(3)

contrasts(dat_helm$Group) |>
  as.data.frame() |>
  tibble::rownames_to_column("Level") |>
  gt() |>
  tab_header(title = "Design Matrix: Helmert Coding")

fit_helm <- lm(Score ~ Group, data = dat_helm)

tidy(fit_helm) |>
  mutate(across(where(is.numeric), ~ round(., 3))) |>
  gt() |>
  tab_header(title = "Coefficient Table: Helmert Coding")
```

- ある水準を、後続水準の平均と順次比較
- 段階的カリキュラムやレベル制指導など、順序を意識した比較に有効

## 6.5 順序尺度にはPolynomial coding

```{r}
#| label: polynomial-coding

set.seed(2026)

dat_ord <- tibble(
  CEFR = factor(rep(c("A1", "A2", "B1", "B2"), each = 80),
                levels = c("A1", "A2", "B1", "B2"),
                ordered = TRUE),
  Score = c(
    rnorm(80, 48, 7),
    rnorm(80, 56, 7),
    rnorm(80, 66, 7),
    rnorm(80, 74, 7)
  )
)

contrasts(dat_ord$CEFR) <- contr.poly(4)

fit_poly <- lm(Score ~ CEFR, data = dat_ord)

tidy(fit_poly) |>
  mutate(across(where(is.numeric), ~ round(., 3))) |>
  gt() |>
  tab_header(title = "Polynomial Contrasts for Ordered CEFR Levels")
```

`CEFR.L`（線形成分）だけでなく、`CEFR.Q`（二次成分）などが有意かどうかを確認することで、伸び方の形状（線形か曲線的か）を検討できます。

## 6.6 コーディング間の不変性チェック

```{r}
#| label: invariance-check

anova(fit_treat)
anova(fit_sum)
anova(fit_helm)

all.equal(unname(fitted(fit_treat)), unname(fitted(fit_sum)))
all.equal(unname(fitted(fit_treat)), unname(fitted(fit_helm)))
```

係数は変わりますが、**同じモデル空間なら予測値は一致**します。したがって、研究質問に合わせて「解釈しやすいコーディング」を選ぶことが合理的です。

# 7. 研究質問に基づくコーディング選択

| 研究質問 | 推奨コーディング | 解釈の中心 |
|---|---|---|
| 介入群を統制群と比較したい | Treatment | 参照群との差 |
| 全体平均からの偏差を見たい | Sum | grand meanとの差 |
| 段階的比較をしたい | Helmert | 逐次的な群比較 |
| 順序因子のトレンドを見たい | Polynomial | 線形・二次・三次成分 |

# 9. 実務上の落とし穴

1. **球面性と複合対称性を混同しない。**
2. **交互作用が有意なのに主効果だけで結論を出さない。**
3. **係数表だけで議論せず、`emmeans`で比較を統一する。**
4. **効果量をp値の代替ではなく補完情報として扱う。**
5. **コーディングの選択理由を本文で明示する。**
