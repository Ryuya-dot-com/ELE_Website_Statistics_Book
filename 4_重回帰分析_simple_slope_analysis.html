6.1 はじめに

本章では、英語教育データを想定して、重回帰分析（lm）と Simple Slope Analysis（交互作用の下位検定）を実装します。 読者は回帰分析の基本を既習として、次の問いに答えられることを目標にします。

  • 学習方法の効果は、学習者の proficiency によって変わるか。
  • その差は、Immediate/Delayed のどちらで大きいか。
  • 交互作用を、係数・図・下位検定で整合的に説明できるか。
この章の到達目標
  • 連続変数を含む交互作用モデルを、理論先行で仕様化できる。
  • emtrends() と emmeans() を使い分けて simple slope を報告できる。
  • 診断、ロバストSE、CVを使って「推定の頑健性」を説明できる。
本章の立ち位置
  • 本章は、lm による回帰モデレーションの基礎訓練です。
  • 反復測定や項目レベルの 0/1 応答を主対象にする場合は、GLMM を優先します (Barr et al., 2013; Jaeger, 2008)。

6.1.1 分析フロー

  1. 研究仮説からモデル式を定義（理論先行）。
  2. z変換とコントラスト設定で係数の意味を明確化。
  3. モデル推定後、前提診断とロバストSEで安定性を確認。
  4. emtrends() で simple slope、emmeans() で条件付き群差を検定。
  5. 図示と報告テンプレートで、結果を再現可能に記述。
初回読者向けの最短ルート（20分）
  1. m_theory を実行して summary() を確認する。
  2. emtrends() で傾き差（simple slope）を確認する。
  3. emmip() で図を見て、方向と大きさを言語化する。
  4. 余力があれば check_model() と HC3 で頑健性を確認する。

最初は「有意かどうか」だけでなく、方向（正/負） と 教育的に意味のある差か を優先して読む。

6.2 統計チュートリアル文献との接続（回帰中心）

本章は、SLA理論の紹介よりも重回帰分析の実装手順に焦点を置きます。
とくに次のチュートリアル文献を実務ガイドとして参照します。

  • 交互作用の下位検定（simple slope / Johnson–Neyman）: Bauer & Curran (2005) (Bauer & Curran, 2005)
  • 実装支援（計算・可視化の運用）: Preacher et al. (2006) (Preacher et al., 2006)
  • 多カテゴリ変数を含む交互作用の扱い: Hayes & Montoya (2017) (Hayes & Montoya, 2017)
  • モデレーション研究の検出力設計: Brown & Abulela (2025) (Brown & Abulela, 2025)

6.2.1 チュートリアル論文の具体例（本章との対応）

  • Bauer & Curran (2005): simple slope だけでなく Johnson-Neyman（J-N）法を使って「有意となるモデレータ範囲」を示す。 本章の emtrends() と J-N 節はこの流れに対応 (Bauer & Curran, 2005)。
  • Preacher et al. (2006): 交互作用のプロービングを計算ツールで実装する実務手順を提示。 本章の「式だけで終わらず、条件付き効果を計算して報告する」方針に直結 (Preacher et al., 2006)。
  • Hayes & Montoya (2017): 3群以上のカテゴリ変数が絡む回帰交互作用で、符号化と条件付き効果比較をどう行うかを整理。 本章では2水準を扱うが、拡張時の設計はこの枠組みに接続 (Hayes & Montoya, 2017)。
  • Baranger et al. (2023): 交互作用検出力の具体例として、相関を入力した解析的/シミュレーションベースの power 分析を提示 （例: r.x1.y, r.x2.y, r.x1.x2, r.x1x2.y）(Baranger et al., 2023)。
  • Brown & Abulela (2025): MMR の power を段階的に組み立てるチュートリアル。 「小さな交互作用」「信頼性」「抑制/増幅」を見込んだ設計を、Rコード付きで示す (Brown & Abulela, 2025)。

本章後半に、以下の4テーマをコード付きレシピとしてまとめます。

  • Bauer/Curran: emtrends() + Johnson-Neyman
  • Preacher: 条件付き効果の計算手順
  • Hayes/Montoya: 多カテゴリ交互作用
  • Baranger/Brown: 交互作用の検出力設計
読み方の実務ルール
  • 交互作用が有意でも、simple slope と予測図を併記しない限り教育的含意は確定しない。
  • 係数の有意性だけでなく、効果量・信頼区間・教育的に意味のある差を同時に報告する。
  • コントラストの設計行列や符号化の厳密な扱いは、第2章 （2_ANOVAとカテゴリカル変数のコーディング.qmd）で確認する。
  • 反復測定・項目効果が中心の設計は、第6章 （6_混合効果モデル_2値ロジスティック回帰.qmd）に接続する (Brysbaert & Stevens, 2018; Cunnings, 2012)。

        )

        }
      }

    }

6.4 変数変換と係数解釈

  • z変換（連続変数）
  • コントラスト（カテゴリカル）
  • モデル式

z変換は、係数を「1 SD 増加あたりの変化量」として読めるようにし、 交互作用の解釈を安定化します(Aiken & West, 1991; Cohen et al., 2003)。

コード 
wide_scores <- wide_scores |>
  mutate(
    Proficiency = as.numeric(scale(Proficiency)),
    Aptitude = as.numeric(scale(Aptitude)),
    Treatment = factor(Treatment, levels = treatments),
    Time = factor(Time, levels = times)
  )

2水準要因を sum coding（-1, +1）にすると、 主効果は「他変数平均化後の平均差」として読みやすくなります。

コード 
contrasts(wide_scores$Treatment) <- contr.sum(2)
contrasts(wide_scores$Time) <- contr.sum(2)

本章の主モデルは次式です。

\[ \begin{aligned} \mathbb{E}[Y] &= \beta_0 + \beta_1\,\text{Trt} + \beta_2\,\text{Time} + \beta_3\,Z_{\text{Prof}} + \beta_4\,Z_{\text{Apt}} \\ &\quad + \beta_5(\text{Trt}\times\text{Time}) + \beta_6(\text{Trt}\times Z_{\text{Prof}}) + \beta_7(\text{Time}\times Z_{\text{Prof}}) \end{aligned} \]

  • \(Y\) は Score_Recall。
  • Trt と Time は sum coding。
  • \(\beta_6\) が「Treatment による Proficiency 傾き差」（主要関心）。

6.9.2 2. Preacher: 条件付き効果の計算手順

Preacher et al. (2006) の流れに合わせ、条件付き効果を係数と分散共分散行列から計算します (Preacher et al., 2006)。

\[ \hat{\\tau}(z)=\hat{\\beta}_{T}+z\hat{\\beta}_{T\\times Z},\quad SE\\{\\hat{\\tau}(z)\\}=\\sqrt{Var(\\hat{\\beta}_{T})+z^2Var(\\hat{\\beta}_{T\\times Z})+2zCov(\\hat{\\beta}_{T},\\hat{\\beta}_{T\\times Z})} \]

コード 
dat_preacher <- wide_scores |>
  mutate(Treatment01 = relevel(factor(Treatment), ref = "Glossing"))

m_preacher <- lm(
  Score_Recall ~ Treatment01 * Proficiency + Time + Aptitude,
  data = dat_preacher
)

calc_conditional_effect <- function(z, model,
                                    b_main = "Treatment01Writing",
                                    b_int = "Treatment01Writing:Proficiency") {
  b <- coef(model)
  V <- vcov(model)
  df <- df.residual(model)

  est <- b[b_main] + z * b[b_int]
  var_est <- V[b_main, b_main] + (z^2) * V[b_int, b_int] + 2 * z * V[b_main, b_int]
  se <- sqrt(var_est)
  t_val <- est / se
  p_val <- 2 * pt(abs(t_val), df = df, lower.tail = FALSE)
  ci <- est + qt(c(0.025, 0.975), df = df) * se

  tibble(
    Proficiency = z,
    Estimate = est,
    SE = se,
    t = t_val,
    p = p_val,
    CI_low = ci[1],
    CI_high = ci[2]
  )
}

simulate_mmr <- function(n, beta_x1 = 0.25, beta_x2 = 0.20, beta_int = 0.10, sigma = 1) {
  x1 <- rnorm(n)
  x2 <- rnorm(n)
  y <- beta_x1 * x1 + beta_x2 * x2 + beta_int * (x1 * x2) + rnorm(n, 0, sigma)
  tibble(y, x1, x2)
}

power_delta_r2 <- function(n, nsim = 150, beta_int = 0.10) {
  p_vals <- replicate(nsim, {
    d <- simulate_mmr(n = n, beta_int = beta_int)
    m0 <- lm(y ~ x1 + x2, data = d)
    m1 <- lm(y ~ x1 * x2, data = d)
    anova(m0, m1)$`Pr(>F)`[2]
  })
  mean(p_vals < 0.05, na.rm = TRUE)
}

6.10 報告テンプレート（英語教育研究向け）

  • モデル: lm(Score_Recall ~ Treatment * Time + Treatment * Proficiency + Time * Proficiency + Aptitude)
  • コーディング: Treatment/Time は sum coding、Proficiency/Aptitude は z 変換。
  • 当てはまり: \(R^2\)、調整 \(R^2\)、AIC/BIC（必要ならCV指標）。
  • 主要効果: 各係数の \(\beta\), SE, t, p, 95% CI。
  • 交互作用: Treatment × Proficiency の係数と、emtrends() による傾き差。
  • 下位検定: Proficiency = -1, 0, +1 での条件付き群差。
  • 診断: 等分散性・正規性・共線性・ロバストSEの結果。
  • 実質解釈: 「高 proficiency 層ほど Writing 効果が増幅」など、教育的意味を1文で明示。

6.10.1 結果解釈テンプレート（理論接続版）

以下の3文セットで書くと、統計結果とSLA理論の接続が明確になります。

  1. 統計的記述: 「Treatment × Proficiency は有意であった（β = …, 95% CI […, …]）。」
  2. 条件付き解釈: 「simple slope では、Proficiency = +1SD で Writing 優位が確認された。」
  3. 理論的含意: 「このパターンは、学習者要因が処置効果を調整する ATI の見立てと整合的である (Li, 2015)。」

補足として、コントラスト設計に踏み込んだ厳密解釈は第2章で扱います （本章では運用上の最小要件に絞ります）。

6.11 実データへ移すときの注意

  • 参加者内反復測定や項目効果が強い場合、lm では独立性仮定が崩れます。
  • 語彙テストの項目正誤（0/1）を扱う場合、logit GLMM が原則です (Barr et al., 2013; Jaeger, 2008)。
  • したがって本章は「交互作用解釈の基礎」と位置づけ、設計に応じて第6章へ接続してください。

## はじめに

本章では、英語教育データを想定して、**重回帰分析（`lm`）**と
**Simple Slope Analysis（交互作用の下位検定）**を実装します。
読者は回帰分析の基本を既習として、次の問いに答えられることを目標にします。

- 学習方法の効果は、学習者の proficiency によって変わるか。
- その差は、Immediate/Delayed のどちらで大きいか。
- 交互作用を、係数・図・下位検定で整合的に説明できるか。

::: {.callout-tip}
### この章の到達目標

- 連続変数を含む交互作用モデルを、理論先行で仕様化できる。
- `emtrends()` と `emmeans()` を使い分けて simple slope を報告できる。
- 診断、ロバストSE、CVを使って「推定の頑健性」を説明できる。
:::

::: {.callout-note}
### 本章の立ち位置

- 本章は、**`lm` による回帰モデレーションの基礎訓練**です。
- 反復測定や項目レベルの 0/1 応答を主対象にする場合は、GLMM を優先します
  [@jaeger2008; @barr2013]。
:::

### 分析フロー

1. 研究仮説からモデル式を定義（理論先行）。
2. z変換とコントラスト設定で係数の意味を明確化。
3. モデル推定後、前提診断とロバストSEで安定性を確認。
4. `emtrends()` で simple slope、`emmeans()` で条件付き群差を検定。
5. 図示と報告テンプレートで、結果を再現可能に記述。

::: {.callout-tip}

### モデル式

本章の主モデルは次式です。

$$
\begin{aligned}
\mathbb{E}[Y] &= \beta_0 + \beta_1\,\text{Trt} + \beta_2\,\text{Time}
+ \beta_3\,Z_{\text{Prof}} + \beta_4\,Z_{\text{Apt}} \\
&\quad + \beta_5(\text{Trt}\times\text{Time})
+ \beta_6(\text{Trt}\times Z_{\text{Prof}})
+ \beta_7(\text{Time}\times Z_{\text{Prof}})
\end{aligned}
$$

- $Y$ は `Score_Recall`。
- `Trt` と `Time` は sum coding。
- $\beta_6$ が「Treatment による Proficiency 傾き差」（主要関心）。

### 2. Preacher: 条件付き効果の計算手順

Preacher et al. (2006) の流れに合わせ、条件付き効果を係数と分散共分散行列から計算します
[@preacher2006]。

$$
\hat{\\tau}(z)=\hat{\\beta}_{T}+z\hat{\\beta}_{T\\times Z},\quad
SE\\{\\hat{\\tau}(z)\\}=\\sqrt{Var(\\hat{\\beta}_{T})+z^2Var(\\hat{\\beta}_{T\\times Z})+2zCov(\\hat{\\beta}_{T},\\hat{\\beta}_{T\\times Z})}
$$

```{r}
dat_preacher <- wide_scores |>
  mutate(Treatment01 = relevel(factor(Treatment), ref = "Glossing"))

m_preacher <- lm(
  Score_Recall ~ Treatment01 * Proficiency + Time + Aptitude,
  data = dat_preacher
)

calc_conditional_effect <- function(z, model,
                                    b_main = "Treatment01Writing",
                                    b_int = "Treatment01Writing:Proficiency") {
  b <- coef(model)
  V <- vcov(model)
  df <- df.residual(model)

  est <- b[b_main] + z * b[b_int]
  var_est <- V[b_main, b_main] + (z^2) * V[b_int, b_int] + 2 * z * V[b_main, b_int]
  se <- sqrt(var_est)
  t_val <- est / se
  p_val <- 2 * pt(abs(t_val), df = df, lower.tail = FALSE)
  ci <- est + qt(c(0.025, 0.975), df = df) * se

  tibble(
    Proficiency = z,
    Estimate = est,
    SE = se,
    t = t_val,
    p = p_val,
    CI_low = ci[1],
    CI_high = ci[2]
  )
}

manual_conditional <- purrr::map_dfr(c(-1, 0, 1), calc_conditional_effect, model = m_preacher)
manual_conditional
```

### 4. Baranger/Brown: 交互作用 power 分析

#### 手順A: 論文値で解析的 power 曲線を作る（`power_interaction_r2`）

以下は、チュートリアル論文で示される代表値を用いた設計例です。

- **Baranger et al. (2023)** の例:
  `r.x1.y=.51, r.x2.y=.22, r.x1.x2=.09` を固定し、
  `r.x1x2.y=.09`（比較的大きい交互作用）と
  `r.x1x2.y=.045`（小さい交互作用）を比較 [@baranger2023]。
- **Brown & Abulela (2025)** の例:
  `r.x1.y=-.18, r.x2.y=-.03, r.x1.x2=.64, r.x1x2.y=.10` を使い、
  信頼性 `rel.x1=.89, rel.x2=.95, rel.y=.79` を反映した場合と、
  完全信頼性（1.0）を比較 [@brownabulela2025]。

#### 手順C: 段階的モデル比較（BrownのMMR手順に対応）

`InteractionPoweR` に加えて、増分説明率（`ΔR²`）に基づく
`anova(m0, m1)` のシミュレーションで、BrownのMMR的な考え方を再現します。

```{r}
simulate_mmr <- function(n, beta_x1 = 0.25, beta_x2 = 0.20, beta_int = 0.10, sigma = 1) {
  x1 <- rnorm(n)
  x2 <- rnorm(n)
  y <- beta_x1 * x1 + beta_x2 * x2 + beta_int * (x1 * x2) + rnorm(n, 0, sigma)
  tibble(y, x1, x2)
}

power_delta_r2 <- function(n, nsim = 150, beta_int = 0.10) {
  p_vals <- replicate(nsim, {
    d <- simulate_mmr(n = n, beta_int = beta_int)
    m0 <- lm(y ~ x1 + x2, data = d)
    m1 <- lm(y ~ x1 * x2, data = d)
    anova(m0, m1)$`Pr(>F)`[2]
  })
  mean(p_vals < 0.05, na.rm = TRUE)
}

set.seed(2024)
power_grid <- tibble(N = seq(120, 320, by = 20)) |>
  mutate(power = purrr::map_dbl(N, power_delta_r2, nsim = 150, beta_int = 0.10))

power_grid |>
  dplyr::filter(power >= 0.80) |>
  dplyr::slice_head(n = 1)
```

## 報告テンプレート（英語教育研究向け）

- モデル: `lm(Score_Recall ~ Treatment * Time + Treatment * Proficiency + Time * Proficiency + Aptitude)`
- コーディング: `Treatment/Time` は sum coding、`Proficiency/Aptitude` は z 変換。
- 当てはまり: $R^2$、調整 $R^2$、AIC/BIC（必要ならCV指標）。
- 主要効果: 各係数の $\beta$, SE, t, p, 95% CI。
- 交互作用: `Treatment × Proficiency` の係数と、`emtrends()` による傾き差。
- 下位検定: `Proficiency = -1, 0, +1` での条件付き群差。
- 診断: 等分散性・正規性・共線性・ロバストSEの結果。
- 実質解釈: 「高 proficiency 層ほど Writing 効果が増幅」など、教育的意味を1文で明示。

### 結果解釈テンプレート（理論接続版）

以下の3文セットで書くと、統計結果とSLA理論の接続が明確になります。

1. **統計的記述**: 「`Treatment × Proficiency` は有意であった（β = ..., 95% CI [..., ...]）。」
2. **条件付き解釈**: 「simple slope では、`Proficiency = +1SD` で Writing 優位が確認された。」
3. **理論的含意**: 「このパターンは、学習者要因が処置効果を調整する ATI の見立てと整合的である
   [@li2015]。」

補足として、コントラスト設計に踏み込んだ厳密解釈は第2章で扱います
（本章では運用上の最小要件に絞ります）。

## 実データへ移すときの注意

- 参加者内反復測定や項目効果が強い場合、`lm` では独立性仮定が崩れます。
- 語彙テストの項目正誤（0/1）を扱う場合、logit GLMM が原則です
  [@jaeger2008; @barr2013]。
- したがって本章は「交互作用解釈の基礎」と位置づけ、設計に応じて第6章へ接続してください。
